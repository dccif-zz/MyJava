#  B站flv视频合并

### 使用方法

```bash
java -jar bilimerge-<版本号>.jar <目标目录> <输出目录>
```

由于使用的是FFmpeg，请确保FFmpeg.exe 和jar文件在同以目录下

目录格式暂时使用Win下格式

例如

- C:\DownFile\18023**
- C:\Out\YouDir

---

### 编译方法

导入项目根目录的build.gradle即可
